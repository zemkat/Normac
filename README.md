Normac
=======

Normac is a system for normalization and access checking of MARC records.  It includes an object-oriented MARC library in PHP.

Files:

* `php/nor` -- normalizer
* `php/mac` -- access checker
* `php/lib` -- MARC library
* `php/etc` -- utility programs
